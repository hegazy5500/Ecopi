# Privacy and Data Minimization

EcoLoop is designed to align with Pi ecosystem guidance that discourages unnecessary personal data collection.

## Data model choices

- Pi identity fields instead of email/password account system
- no requirement for phone number
- no requirement for external social login
- transactions intended to remain inside Pi-native flow

## Recommended production policy

Collect only what is needed for:
- user authentication through Pi
- recycling uploads
- rewards history
- marketplace operations
- NFT metadata creation

Avoid collecting:
- unnecessary contact details
- unrelated personal profile fields
- duplicate credentials already managed by Pi
